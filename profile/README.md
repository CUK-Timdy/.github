##  서비스 소개

**가마니(Gamani)** 는  
대학생의 학습 흐름이 여러 서비스에 흩어져 있다는 문제에서 출발한 프로젝트입니다.

스터디 모집은 커뮤니티에서,  
공부 시간 측정은 타이머 앱에서,  
일정 관리는 플래너 앱에서 따로 하다 보면  
학습 흐름이 끊기고 스터디도 일회성으로 끝나기 쉽습니다.

가마니는 이런 문제를 해결하기 위해,  
**스터디 매칭 · 학습 기록 · 일정 관리 · 랭킹 요소**를  
하나의 앱 안에서 통합 제공하는 **캠퍼스 기반 학습 플랫폼**입니다.

---

##  Why Gamani?

- 학교 인증 기반으로 **신뢰도 있는 스터디 환경**
- 같은 학교 / 학과 학생들과의 **자연스러운 연결**
- 학과·학교 단위 랭킹을 통한 **학습 동기부여**
- 타이머, 플래너, 스터디를 하나로 묶은 **올인원 학습 경험**

##  주요 기능

### 👥 Study

- 그룹 생성 / 참여 / 관리
- 투표형 일정 조율
- 타임블록 히트맵 기반 시간 조정

### ⏱ Timer

- 개인별 공부 시간 측정 및 기록
- 과목별 공부 시간 관리
- 강의시간표 연동 과목 선택
- 전공별 · 학과별 랭킹 제공

### 📅 Planner

- 과목별 TODO 관리
- 달성률 표시
- D-Day 등록
- 복기 노트 작성
