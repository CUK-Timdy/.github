# CUK-Timdy

가톨릭대학교 기반 학습 경험을 만드는 팀 **Timdy**입니다.  
작은 기능을 빠르게 만들고, 실제 사용자 흐름 안에서 계속 다듬습니다.

## What We Build

- 캠퍼스 기반 학습 플랫폼
- 스터디/기록/계획을 한 흐름으로 잇는 제품
- iOS 중심의 빠른 실험과 배포

## Current Project
